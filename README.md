# Dotfiles
This is my personal configuration for all my envioroments (Windows, OSX and Linux)

## Requirements
- Nvim
- Zsh
- Fzf
- [Colorls](https://github.com/athityakumar/colorls?tab=readme-ov-file#installation)

## Installation
Run the following command:

```bash
./setup.sh
```
